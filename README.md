# Singara

Singara is a mobile-first beauty marketplace connecting verified makeup artists with customers for events such as weddings, parties, editorial shoots, and corporate functions.

The platform helps customers easily discover and book professional makeup artists while enabling artists to receive structured booking requests and grow their clientele.

This project is the **V1 frontend implementation** of the Singara marketplace.

---

# Product Vision

Singara aims to become the trusted platform where beauty professionals and customers connect seamlessly.

The platform focuses on:

- Verified makeup artists
- Structured bookings
- Secure payments
- Event-based discovery
- Data capture for future AI beauty recommendations

The long-term vision is to build a **beauty intelligence platform powered by structured makeup and style data**.

# User Types

The platform supports three primary roles:

### Customer

Customers can:

- Discover makeup artists
- Filter artists
- View artist portfolios
- Book artists for events
- Manage bookings

### Artist

Artists can:

- Apply for verification
- Receive booking requests
- Manage bookings
- Upload portfolio
- Track earnings

### Admin

Admins can:

- Review artist applications
- Approve or reject artists
- Manage disputes
- Monitor platform activity

---

# Core Features

## Customer Flow

Customers can:

1. Select event type
2. Select location
3. Select budget
4. Optionally choose skin type and tone
5. View recommended artists
6. Explore artist profiles
7. Book artist
8. Confirm booking

---

## Artist Features

Artists have access to:

- Artist dashboard
- Booking request management
- Portfolio management
- Availability updates
- Earnings overview
- Referral system

---

## Escrow Payment System (Planned)

The platform will implement escrow payments where:

- Customer pays full amount upfront
- Singara charges **15% commission**
- Payment is released to artist after service completion

---

## Data Capture for AI

Each booking captures structured data for future AI capabilities.

Captured data includes:

- Event type
- Budget
- Location cluster
- Skin tone
- Skin type
- Style preference
- Artist ID
- Products used
- Before photo
- After photo
- Customer rating
- Repeat booking flag

This dataset will power **future AI beauty recommendations**.
